Daily Theme with Bootstrap Framework

An example of data entry form with daily theme and with Bootstrap frameworks.
The site.css file used in this website contains CSS codes to overwrite colors used in the Bootstrap Frameworks.
This website is using Jquery 3.4.1 and CSS variables (-- not test with old versions of Jquery).
For example of changing color in Jquery with CSS variables.
$(selector).css("background-color", "rgb(var(--todayColor))");
